Aprendendo DOM

É uma API que permite manipularmos o html com js. O doc. HTML é visto como "document" e terá associado a ele vários métodos.

  1. getElementById("nome")-pega o elemento prlo id
  2. innerHTML - inserir ou mudar conteúdo
  3. Mudar estilo
  4. ClassList - Insere uma classe ao elemento HTML
    • .add - adiciona uma classe ao elemento.
    • .remove - remove uma classe do elemento.
    • .toggle - insere ou remove a classe se tiver o elemento
  5. CreateElement - cria elementos
  6. querySelectorAll - Seleciona vários elementos ao mesmo tempo.
