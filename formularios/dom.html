API auxiliar no desenvolvimento de aplicações, simplifica forma de acessar doc HTML.

DOM é uma interface de programação para documentos HTML e XML. Ele representa a página para que os programas possam alterar a estrutura do documento, estilo e conteúdo. O DOM fornece uma representação estruturada do documento como uma árvore de nós e objetos. Dessa forma, os desenvolvedores podem criar scripts que alteram a estrutura do documento, estilo e conteúdo. O DOM não é parte do JavaScript, mas o JavaScript pode ser usado para acessar e manipular o DOM.

DOM: API para manipular dados

Ex: document.getElementById("id") - acessa elemento pelo id

Ex: document.getElementsByClassName("classe") - acessa elementos pela classe
